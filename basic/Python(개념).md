# Python

* Object Oriented Programming

* Dynamic typing language

  * 자료형을 정의하지 않음

  ```python
  greeting = "Hello world"
  print(greeting) # Hello world # 문자
  greeting = 1
  print(greeting) # 1 # 숫자
  greeting = True
  print(greeting) # True # boolean
  ```
