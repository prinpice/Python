

# 주석(Comment)

* 코드 실행시에 실행되지 않음

* `#`, `" "`, `' '`로 표현함

* `docstring`은 `""" """`, `''' '''`으로 표현함

  : 여러 줄의 주석을 작성할 수 있으며, 일반적으로 함수/클래스 선언 후에 해당하는 설명을 위해 활용함

  ```python
  in : 
      # 이 줄을 실행되지 않습니다.
  	def mysum(a, b):
      	"""이것은 덧셈함수입니다.
      	이 줄도 실행되지 않습니다."""
          
      mysum.__doc__ # 주석을 보여주는 함수
      
  out : 
      '이것은 덧셈함수입니다.\n    이 줄도 실행되지 않습니다.'
  ```
