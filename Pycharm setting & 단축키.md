## 단축키

* `alt+enter` : 에러잡아줌
* `alt+shift+커서` : 줄이동
* `alt+f12` : terminal
* `ctrl+alt+r` : manage.py 접속
* `ctrl+alt+l` : 줄정렬
* `ctrl+d` : duplicate(커서있는 줄을 copy&paste)
* `ctrl+g` : 
* `shift+enter` : 줄바꿈
* `shift 2번` : 파일 조회
